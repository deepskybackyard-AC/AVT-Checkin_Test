# AVT Check-in Demo 0.2.0-test.9

Ein-Gerät-Testversion mit weiter komprimierter Veranstaltungsanzeige für den Check-in.

## Neu in dieser Version

- Der obere Veranstaltungsblock ist jetzt deutlich kompakter.
- Sichtbar sind nur noch:
  - eingecheckte Personen
  - sicher freie Plätze
- Rechts daneben befindet sich ein Info-Button mit dem bereitgestellten Symbol.
- Über diesen Button werden die Veranstaltungsdetails in einem Dialog angezeigt.
- Dadurch bleibt der Bereich unterhalb der fünf Hauptbuttons freier, sodass der Check-in-Button in der Regel schneller sichtbar ist.
